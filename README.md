# Ujjwal Pandey - Frontend Developer Portfolio

Welcome to my personal portfolio repository! This is a modern, highly interactive, and responsive web application built to showcase my skills, experience, and projects as a Senior Frontend Developer.

## 🚀 Live Demo
[https://ujjwal-pandey-portfolio.netlify.app/](https://ujjwal-pandey-portfolio.netlify.app/)

## 🛠️ Built With

This portfolio is built using modern web development technologies to ensure performance, scalability, and stunning visuals:

- **[React.js](https://reactjs.org/)** - Frontend library for building user interfaces.
- **[Vite](https://vitejs.dev/)** - Next Generation Frontend Tooling for fast development.
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework for rapid UI development.
- **[Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction)** - For rendering beautiful 3D elements and interactive canvases.
- **[Framer Motion](https://www.framer.com/motion/)** - A production-ready motion library for React to handle smooth animations and page transitions.
- **[EmailJS](https://www.emailjs.com/)** - Client-side email sending functionality for the contact form.

## ✨ Features

- **3D Interactive Elements**: Custom 3D models integrated using Three.js to provide an immersive user experience.
- **Responsive Design**: fully optimized for all screen sizes (mobile, tablet, desktop) using Tailwind CSS.
- **Smooth Animations**: Element revealing, scrolling, and hover effects powered by Framer Motion.
- **Dynamic Content**: Sections for Experience, Projects, Tech Stack, and Contact.
- **Contact Form**: Functional contact form powered by EmailJS that sends messages directly to my email.

## ⚙️ Local Development Setup

To get a local copy up and running, follow these simple steps:

### Prerequisites

You need to have Node.js installed on your machine.
- [Node.js (LTS version recommended)](https://nodejs.org/en/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
   *(Note: if you see dependency warnings, you can also try running `npm install --legacy-peer-deps` or `npm audit fix`)*

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in Browser**
   Open your browser and navigate to `http://localhost:5173/` (or the port specified in your terminal) to view the app.

## 📂 Project Structure

```text
src/
├── assets/          # Images, logos, and icons
├── components/      # Reusable React components (Navbar, Hero, About, etc.)
│   └── canvas/      # 3D canvas components (Three.js/Fiber)
├── Constants/       # Centralized data (experience, projects, navLinks)
├── hoc/             # Higher-Order Components (e.g., SectionWrapper for animations)
├── utls/            # Utility functions (motion configurations)
├── App.jsx          # Main application routing and component assembly
├── main.jsx         # React application entry point
├── Index.css        # Global CSS styles and Tailwind directives
└── styles.js        # Reusable Tailwind class configurations
```
